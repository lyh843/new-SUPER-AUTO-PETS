# Super Auto Pets - 运行指南

## 🚀 如何运行项目

### 推荐方式：使用 Qt Creator

1. **打开 Qt Creator**
2. **打开项目**：
   - 点击 `文件` → `打开文件或项目`
   - 选择项目根目录中的 `CMakeLists.txt` 文件
3. **配置项目**：
   - 选择 Kit（构建套件），推荐使用 MinGW 64-bit
   - 点击 `Configure Project`
4. **构建并运行**：
   - 点击左下角的绿色 ▶️ 按钮（或按 `Ctrl+R`）
   - 等待编译完成，程序会自动运行

### 问题排查

如果遇到编译错误，请尝试：

1. **清理构建**：
   - 在 Qt Creator 中：`构建` → `清理全部`
   - 然后再次点击 ▶️ 运行

2. **重新配置 CMake**：
   - 删除 `build` 目录
   - 重新打开项目

3. **检查 Qt 版本**：
   - 确保安装了 Qt 6.10.0 或更高版本
   - 确保安装了 MinGW 编译器

## ✨ 完整功能说明

### 1. 开始界面
程序启动后会显示主菜单，包含三个按钮：

- **🎯 游玩** - 开始新游戏
- **📚 宠物图鉴** - 查看所有宠物和食物信息  
- **📊 战绩** - 查看历史游戏记录

### 2. 商店系统
- 购买宠物（3金币）
- 购买食物强化宠物
- 出售宠物（获得1金币）
- 刷新商店（1金币）
- 相同宠物自动合并升级

### 3. 战斗系统 ⚔️
- **AI对手生成**：根据回合数自动调整难度
- **回合制战斗**：双方轮流攻击
- **战斗模式**：
  - ⏩ 自动战斗：快速完成战斗
  - ⏯️ 单步执行：逐步观看战斗过程
- **战斗日志**：实时显示所有战斗事件
- **特殊效果**：
  - 🛡️ 护甲：减少伤害
  - 🍉 护盾：50%减伤
  - 🍄 复活：死亡时复活一次
- **战斗结果**：
  - 胜利：+1 奖杯
  - 失败：-1 生命
  - 平局：无变化

### 4. 游戏目标
- **胜利**：获得5个奖杯 🏆
- **失败**：生命值降至0 ❤️

### 5. 宠物图鉴
- 显示所有8种宠物的详细信息
- 显示所有9种食物的详细信息
- 包含攻击力、生命值、技能描述等

### 6. 战绩系统
- 查看历史游戏记录
- 显示统计信息汇总
- 可清空所有记录

## 🎮 游戏流程

```
开始界面

    ↓
 新回合/游戏结束
```
